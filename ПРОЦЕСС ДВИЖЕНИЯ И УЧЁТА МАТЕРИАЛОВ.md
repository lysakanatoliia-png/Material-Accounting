**ПРОЦЕСС ДВИЖЕНИЯ И УЧЁТА МАТЕРИАЛОВ**

**ПРИНЦИПЫ УЧЁТА МАТЕРИАЛОВ**

• Каждый объект является отдельным складом.

• Каждый автомобиль является отдельным складом.

• Центральный склад — дополнительный статический склад.

• Материалы могут поступать на объект: от поставщика; со склада автомобиля; с центрального склада (напрямую, минуя автомобиль).

• Материалы с объекта могут возвращаться на склад автомобиля, а также перемещаться с автомобиля на центральный склад.

• Нормативный остаток склада автомобиля не используется для комплектации объекта.

• Материалы, имеющиеся на центральном складе, также учитываются и исключаются из объёма закупки при комплектации объекта.

• Данные учёта материалов используются для расчёта фактической себестоимости и юнит-экономики объекта.

**S0. ПОДГОТОВКА К ЗАПУСКУ СИСТЕМЫ МАТЕРИАЛЬНОГО УЧЁТА**

Ответственный: Помощник / бухгалтер

Участники:

• Инженер-ассистент

• Montageleiter

Помощник / бухгалтер:

• формирует единый справочник материалов компании — на основе анализа фактических накладных и закупок за последний отчётный период;

• присваивает код каждой материальной позиции (коды формируются автоматически учётной программой при внесении позиции);

• определяет единицы измерения для каждого материала;

• формирует типовую спецификацию склада автомобиля;

• определяет нормативное количество материалов для комплектации автомобиля;

• разрабатывает формы движения материалов: «автомобиль ↔ объект» (для монтажной бригады), «автомобиль ↔ центральный склад» и «центральный склад → объект» (для Montageleiter);

• создаёт в учётной системе склады автомобилей;

• создаёт в учётной системе центральный склад;

• создаёт шаблон склада объекта;

• проверяет возможность учёта прихода, расхода и перемещения материалов;

• проверяет корректность отображения остатков в системе.

Инженер-ассистент:

• формирует наглядный визуальный каталог материалов (с фотографиями, сгруппированными по типам и диаметрам) на основе справочника, кодов и единиц измерения, подготовленных Помощником/бухгалтером.

Montageleiter:

• разрабатывает и описывает перечень минимально допустимых остатков материалов (позиции и количество) для склада автомобиля;

• согласовывает типовую комплектацию автомобилей;

• проверяет удобство использования форм движения материалов — формы «автомобиль ↔ объект» для монтажной бригады, а также форм «автомобиль ↔ центральный склад» и «центральный склад → объект», которые будет заполнять сам.

**M5. ФИКСАЦИЯ ДВИЖЕНИЯ МАТЕРИАЛОВ МЕЖДУ ОБЪЕКТОМ И СКЛАДОМ АВТОМОБИЛЯ**

Старт события

Во время выполнения работ на объекте используются материалы со склада автомобиля, а после завершения работ монтажная бригада передаёт форму движения материалов с указанием, какие материалы и в каком количестве были взяты со склада автомобиля, а также какие материалы и в каком количестве были возвращены на склад автомобиля после завершения объекта.

Монтажная бригада:

• фиксирует движение материалов между складом автомобиля и объектом;

• фиксирует материалы, взятые со склада автомобиля для выполнения работ;

• фиксирует неиспользованные материалы, возвращённые на склад автомобиля после завершения работ;

• заполняет форму движения материалов;

• передаёт форму Montageleiter для проверки.

Montageleiter:

• проверяет корректность заполнения формы;

• сверяет зафиксированные материалы с фактически выполненными работами;

• подтверждает достоверность данных;

• передаёт подтверждённую форму в офис (Bitrix) для дальнейшего учёта.

➡️ M5a

**M5a. ФИКСАЦИЯ ДВИЖЕНИЯ МАТЕРИАЛОВ ЧЕРЕЗ ЦЕНТРАЛЬНЫЙ СКЛАД**

Ответственный: Montageleiter

Участники:

• Помощник / бухгалтер

Карточка охватывает два типа движения материалов, связанных с центральным складом.

**1. Движение «автомобиль ↔ центральный склад»**

Старт события — одно из событий:

• автомобиль передаётся другому сотруднику или сдаётся перед отпуском — проводится инвентаризация (полная сверка) автомобиля;

• автомобиль приезжает на базу и выгружает излишки материалов сверх нормативной комплектации на центральный склад;

• при комплектации объекта (этап M2) выявлена нехватка на складе автомобиля, которую можно закрыть материалами с центрального склада.

Montageleiter:

• ведёт учёт центрального склада;

• при передаче/сдаче автомобиля проводит инвентаризацию — сверяет фактический остаток с документальным;

• организует перемещение излишков материалов со склада автомобиля на центральный склад;

• организует перемещение материалов с центрального склада на склад автомобиля при необходимости пополнения;

• заполняет форму движения материалов (тип «автомобиль ↔ центральный склад»).

**2. Движение «центральный склад → объект» (напрямую, минуя автомобиль)**

Старт события

На этапе M2 определено, что часть материалов должна быть получена с центрального склада напрямую на объект (например, отправкой почтой/курьером), минуя склад автомобиля.

Montageleiter:

• организует отправку материалов с центрального склада на объект;

• заполняет форму движения материалов (тип «центральный склад → объект»).

Помощник / бухгалтер (для обоих типов движения):

• проверяет корректность заполнения формы;

• вносит перемещение в учётную систему;

• обновляет остатки соответствующих складов (центрального, автомобиля и/или объекта — в зависимости от типа перемещения).

**M6. СВЕРКА ДВИЖЕНИЯ МАТЕРИАЛОВ И ОБНОВЛЕНИЕ ОСТАТКОВ СКЛАДА АВТОМОБИЛЯ**

➡️ M7

**M7. КОНТРОЛЬ МИНИМАЛЬНЫХ ОСТАТКОВ**

Ответственный: Помощник / бухгалтер

Участники:

• Инженер-ассистент

• Montageleiter

Старт события

Обновлены фактические остатки склада автомобиля после завершения учёта материалов по объекту.

Помощник / бухгалтер:

• проверяет фактические остатки материалов склада автомобиля;

• сравнивает фактические остатки с нормативными остатками;

• определяет позиции, которые находятся ниже установленного норматива;

• проверяет наличие дефицитных позиций на центральном складе;

• формирует перечень материалов для пополнения, разделяя его на две части: доступно на центральном складе / отсутствует на центральном складе.

Для позиций, доступных на центральном складе:

• передаёт информацию Montageleiter для организации перемещения (см. M5a, раздел «автомобиль ↔ центральный склад»).

Для позиций, отсутствующих на центральном складе:

• передаёт информацию Инженеру-ассистенту для закупки у поставщика.

Инженер-ассистент:

• получает перечень материалов для закупки (часть, которую не покрывает центральный склад);

• проверяет потребность в закупке;

• согласовывает перечень материалов для пополнения склада автомобиля.

➡️ M8

**M8. ПОПОЛНЕНИЕ СКЛАДА АВТОМОБИЛЯ ОТ ПОСТАВЩИКА**

Ответственный: Инженер-ассистент

Участники:

• Помощник / бухгалтер

• Montageleiter

• Поставщик

Старт события

От Помощника/бухгалтера получен перечень материалов, которые не удалось закрыть с центрального склада (см. M7) — остаток которых ниже нормативного уровня.

Инженер-ассистент:

• анализирует перечень материалов для пополнения;

• определяет необходимый объём закупки;

• формирует заказ поставщику;

• контролирует сроки поставки;

• информирует Montageleiter о запланированном пополнении склада автомобиля.

Поставщик:

• комплектует заказ;

• отгружает материалы;

• формирует расходные документы;

• передаёт документы в систему учёта.

Montageleiter:

• организует получение материалов для склада автомобиля;

• контролирует фактическое поступление материалов;

• сообщает об отсутствующих или повреждённых позициях.

Помощник / бухгалтер:

• получает расходные документы;

• оприходует материалы на соответствующий склад автомобиля;

• обновляет фактические остатки склада автомобиля;

• контролирует достижение нормативных остатков.
