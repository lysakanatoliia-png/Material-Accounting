**ПРОЦЕСС ДВИЖЕНИЯ И УЧЁТА МАТЕРИАЛОВ**

**ПРИНЦИПЫ УЧЁТА МАТЕРИАЛОВ**

• Каждый объект является отдельным складом.

• Каждый автомобиль является отдельным складом.

• Материалы могут поступать на объект:

• от поставщика;

• со склада автомобиля.

• Материалы могут возвращаться только на склад автомобиля.

• Нормативный остаток склада автомобиля не используется для комплектации объекта.

• Данные учёта материалов используются для расчёта фактической себестоимости и юнит-экономики объекта.

**S0. ПОДГОТОВКА К ЗАПУСКУ СИСТЕМЫ МАТЕРИАЛЬНОГО УЧЁТА**

Ответственный: Инженер-ассистент

Участники:

• Помощник / бухгалтер

• Montageleiter

Инженер-ассистент:

• формирует единый справочник материалов компании;

• присваивает код каждой материальной позиции;

• определяет единицы измерения для каждого материала;

• формирует типовую спецификацию склада автомобиля;

• определяет нормативное количество материалов для комплектации автомобиля;

• определяет минимально допустимые остатки по каждой позиции;

• разрабатывает форму движения материалов для монтажной бригады.

Помощник / бухгалтер:

• создаёт в учётной системе склады автомобилей;

• создаёт шаблон склада объекта;

• проверяет возможность учёта прихода, расхода и перемещения материалов;

• проверяет корректность отображения остатков в системе.

Montageleiter:

• проверяет перечень типовых материалов склада автомобиля;

• предоставляет предложения по нормативным остаткам;

• согласовывает типовую комплектацию автомобилей;

• проверяет удобство использования формы движения материалов монтажной бригадой.

**M2. ЗАКУПКА И ПОСТАВКА МАТЕРИАЛОВ**

Ответственный: Инженер-ассистент

Участники:

• Montageleiter

• Помощник / бухгалтер

Старт события

Завершено планирование потребности в материалах.

Инженер-ассистент:

• проверяет фактические остатки материалов на складах автомобилей;

• определяет избыточные остатки материалов сверх нормативной комплектации автомобиля;

• принимает решение об использовании избыточных материалов для обеспечения объекта;

• уменьшает объём закупки на количество доступных избыточных материалов;

• формирует финальный заказ поставщику;

• определяет перечень материалов, которые должны быть взяты со склада автомобиля;

• передаёт Montageleiter информацию относительно:

• материалов, которые необходимо получить от поставщика;

• материалов, которые необходимо использовать со склада автомобиля;

• контролирует своевременность поставки материалов на объект.

Montageleiter:

• подтверждает возможность использования избыточных материалов со склада автомобиля;

• организует передачу материалов со склада автомобиля на объект;

• контролирует наличие материалов для непрерывного выполнения работ.

Помощник / бухгалтер:

• получает от поставщиков расходные накладные;

• проверяет наличие привязки документов к соответствующему объекту;

• готовит документы для дальнейшего оприходования материалов на склад объекта.

Правило использования материалов со склада автомобиля

Материалы со склада автомобиля могут быть использованы для обеспечения объекта только в пределах фактического излишка сверх нормативного остатка, определённого типовой спецификацией склада автомобиля.

Нормативный остаток склада автомобиля не используется для комплектации объекта и должен поддерживаться постоянно.
